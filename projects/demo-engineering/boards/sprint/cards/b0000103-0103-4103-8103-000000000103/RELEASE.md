# Release v0.2.0-demo

## Added

- Sample demo projects in `projects/demo-marketing` and `projects/demo-engineering`
- Vault attachments: markdown, JSON, canvas, SVG

## Changed

- Board columns use theme color tints (Trello-style)

## How to load demos

1. Push `projects/demo-*` to GitHub
2. Run `node scripts/migrate-github-to-supabase.mjs`
3. Open projects in the dashboard app
